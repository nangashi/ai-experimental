# Scoring Rubric

## 1. 検出判定基準

| 記号 | 判定 | スコア | 基準 |
|-----|------|-------|------|
| **○** | 検出 | 1.0 | 正解キーの検出判定基準を **すべて** 満たす指摘がある |
| **△** | 部分検出 | 0.5 | 関連するカテゴリの指摘はあるが、核心的な問題点を正確に捉えていない |
| **×** | 未検出 | 0.0 | 当該問題に関する指摘が全くない |

**△の判定例**: 関連カテゴリの指摘はあるが具体的な問題箇所が異なる、問題の存在は指摘しているが根本原因の分析が不十分

---

## 2. スコア計算式

### 基本スコア（per document, per run）
```
run_score = Σ(検出スコア) + (ボーナス件数 × 0.5) - (ペナルティ件数 × 0.5)
```

### 統合スコア（cross-document, 4 runs）
```
overall_mean = average(doc1_run1, doc1_run2, doc2_run1, doc2_run2)
overall_sd = stddev(doc1_run1, doc1_run2, doc2_run1, doc2_run2)
cross_doc_gap = |doc1_mean - doc2_mean|
```

### ボーナス・ペナルティ

| 種別 | 条件 | スコア | 上限 |
|-----|------|-------|------|
| ボーナス | 正解キーに未掲載かつ perspective.md の評価スコープに合致する事実に基づいた有益な追加指摘 | +0.5/件 | 5件 |
| ペナルティ | スコープ外の指摘、事実に反する指摘、または明らかに誤った分析 | -0.5/件 | なし |

- スコープ境界の判定は perspective.md のボーナス/ペナルティ判定指針に従う
- 判断に迷う場合はペナルティを付与しない（疑わしきは罰せず）

---

## 3. カテゴリバランス指標

### カテゴリ別検出率
```
category_rate = (Σ○ + 0.5 × Σ△) / (category_questions × total_runs)
```

### バランス指標
```
balance = min(category_rates) / max(category_rates)    # 0.0 - 1.0
```

### 回帰検出
カテゴリ検出率が現在ベストから **0.15 以上低下** した場合を「回帰」と判定する。

---

## 4. 推奨判定基準

### 回帰加重差分
```
adjusted_diff = raw_mean_diff - 1.5 × Σ(regressed_category_rate_drops)
```

### 推奨判定

| 条件 | 判定 |
|------|------|
| adjusted_diff > 1.0pt AND cross_doc_gap < 1.5pt AND 回帰なし | バリアントを **推奨** |
| adjusted_diff 0.5〜1.0pt AND 回帰なし | SDが小さい方を **推奨** |
| adjusted_diff < 0.5pt | **現在ベストを推奨** |
| 回帰あり（1カテゴリ以上） | **現在ベストを推奨**（回帰理由を付記） |

### 収束判定

| 条件 | 判定 |
|------|------|
| 3ラウンド連続で改善幅 < 0.5pt | 「収束の可能性あり」 |
| それ以外 | 「継続推奨」 |

---

## 5. ベースラインスコアの取得

固定テスト文書を使用するため、現在ベストのスコアは再評価せず history.md のキャッシュ値を使用する。

---

## 6. テスト文書パラメータ

| パラメータ | 値 |
|-----------|-----|
| テスト文書数 | 2（異なるドメイン） |
| 文書あたりの問題数 | 10問 |
| 深刻度分布 | 重大 3 / 中 4 / 軽微 3 |
| カテゴリ分布 | 主要観点 6-7 / 隣接領域 2 / 微妙な問題 1-2 |
| ボーナス問題候補 | 3-5件/文書 |
| 評価回数 | 2回/文書/バリアント |

---

## 7. 安定性閾値

| 標準偏差 (SD) | 判定 |
|--------------|------|
| SD ≤ 0.5 | 高安定 |
| 0.5 < SD ≤ 1.0 | 中安定 |
| SD > 1.0 | 低安定 |
