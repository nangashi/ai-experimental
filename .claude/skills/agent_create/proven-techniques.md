## 4. ベースライン構築ガイド（新規作成時）

新規エージェント定義を生成する際は、以下の原則を適用してベースラインの初期品質を最大化する:

1. **指示文は英語で記述する** — 全観点で+1.5pt以上の効果と完璧な安定性を達成
2. **評価/分析項目はテーブルまたはマトリクス構造を使う** — 網羅的カバレッジを強制
3. **エージェントの主要タスクに対する明示的指示を含める** — 曖昧さを排除
4. **出力形式はセクション名+簡潔な説明に留める** — 詳細テンプレートは避ける
5. **ドメイン固有の知識がある場合はチェックリストとして含める** — 3-4領域まで
6. **Few-shot例は0-2個に留める** — 入れすぎると注意バイアス発生
7. **重み付け採点・敵対的視点・厳格な分類は避ける** — 実証済みの逆効果パターン
