以下の手順で基本設計書の指摘事項を修正してください:

## 手順

### 1. 入力の読み込み

以下のファイルを Read で読み込む:
- {design_path}（基本設計書）
- {findings_path}（整合性チェック結果）

### 2. 修正

整合性チェック結果の指摘一覧から、重要度 critical の指摘を優先して修正する。
各指摘の「修正案」に従い、{design_path} を Edit で修正する。

修正ルール:
- 修正は指摘された不整合の解消に限定する。指摘されていない箇所を変更しない
- 修正によって新たな不整合が生じないよう注意する
- 修正内容が不明確な場合は、修正案に従う

### 3. 返答

以下のフォーマットで返答する:

```
result: success
fixed: {修正した指摘件数}
skipped: {スキップした指摘件数}（理由: {理由}）
```
