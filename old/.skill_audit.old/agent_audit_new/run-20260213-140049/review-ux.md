- [不可逆操作のガード: perspective 再生成時の上書き]: [Phase 0 Step 5] perspective 自動生成で重大な問題または改善提案がある場合、Step 3 で生成した perspective-source.md を無警告で上書き再生成する。初回生成内容が失われるため、再生成前に AskUserQuestion でユーザー確認を行うか、初回生成内容を別ファイルに保存することを推奨 [impact: low] [effort: low]

- [承認粒度: Phase 3 失敗時の選択肢が複数の処理パターンを含む]: [Phase 3] いずれかのプロンプトで成功結果が0回の場合、「再試行/該当プロンプトを除外して続行/中断」の3択を一度に提示している。再試行と除外はワークフローへの影響が異なるため、「まず再試行するか」→（再試行後も失敗なら）「除外して続行するか中断するか」の2段階確認にすることで、ユーザーがより適切な判断を行える [impact: low] [effort: medium]
