# タスク

情報を収集して蓄積してAIの動作改善に繋げる仕組みを考えて提案して。収集する情報を指示したらその情報がリポジトリに追加されて、AIの行動に反映するのが望ましいです。

## 成果ベース解釈

Claude Codeの既存メカニズム（CLAUDE.md + .claude/instructions/）を活用し、ユーザーの自然言語指示で知識を蓄積→自動反映するフィードバックループを設計する。新しいスキルやツールの開発ではなく、既存の仕組みの上に乗る運用設計を目指す。

## 介入トリガー評価

- 仕様の未決定性: 該当 — カテゴリ体系、自動化レベル、メンテナンス方式に複数の妥当な選択肢
- 検証手段の不在: 該当 — 設計の良さは主観的で形式検証不可
- 不確実性の累積: 非該当 — 設計提案タスクで推測チェーンなし
- 介入頻度: 完了時レビュー
